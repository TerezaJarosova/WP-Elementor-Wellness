# WP-Elementor-Wellness

# Wellness Website – WordPress + Elementor

A modern wellness-focused website designed in Figma and implemented in WordPress using Elementor.  
This project serves as a portfolio case study demonstrating UI/UX design, layout structuring, and WordPress development skills.

---

## 🌿 Live Website

🔗 https://project.theas.design/wellness/

---

## 🧠 Overview

This is a concept wellness website focused on calm visual design, clear structure, and a smooth user experience.  
The project was created as a **showcase piece for portfolio purposes**, with emphasis on design consistency and front-end implementation.

The goal was to translate a Figma design into a fully functional WordPress website using Elementor while maintaining visual harmony and responsiveness across devices.

---

## 🛠 Technologies

- WordPress
- Elementor
- Figma
- Custom CSS (minor adjustments)

---

## ✨ Key Features

- Clean, minimal wellness-oriented UI
- Fully responsive layout (desktop, tablet, mobile)
- Structured content sections with clear hierarchy
- Design-to-development consistency (Figma → WordPress)
- Elementor-based page building

---

## 🎨 Design Approach

The design is based on three core principles:

- **Calmness** – soft visual language and spacing
- **Clarity** – clear structure and readable layout
- **Flow** – intuitive user navigation through sections

---

## 🔁 Process

1. Research and moodboard creation
2. Wireframes in Figma
3. High-fidelity UI design in Figma
4. WordPress setup and configuration
5. Elementor implementation
6. Responsive optimization for all devices
7. Final visual refinement
